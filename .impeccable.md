## Design Context

### Users
Educators, school administrators, and the Diagonally founding team (Mike, Scott, Barbara). They use the admin editor to write blog posts, manage content pages, and publish updates. Context: working from laptops, often in the evening after school hours, writing build-in-public updates and educational content.

### Brand Personality
Inventive, warm, confident. Diagonally is an edtech startup that believes students should build games, not take tests. The brand is optimistic but grounded in real results.

### Aesthetic Direction
**Admin editor:** Substack-like — editorial, warm, writerly. The writing experience should feel like a refined word processor, not a developer tool. Typography-forward, warm tones in the dark theme, generous spacing that lets the content breathe. The toolbar should feel like instruments in a craftsperson's workshop, not a cluttered control panel.

**Public site:** "Cosmic Editorial" — space-themed dark hero sections with constellation animations transitioning to warm cream content areas. Bricolage Grotesque display + Nunito body.

**Anti-references:** Generic Bootstrap admin panels, cluttered WordPress dashboards, anything that looks like a template.

### Design Principles
1. **Content is the hero** — every UI element exists to serve the writing, not to show off
2. **Warm dark, not cold dark** — the admin dark theme should feel like a cozy writing desk at night, not a terminal
3. **Progressive disclosure** — start simple, reveal power through interaction
4. **Intentional typography** — type hierarchy creates the structure, not borders and boxes
5. **Substack-quality polish** — if it doesn't feel as good as Substack's editor, it's not done
